# VS Code Extensions Setup

This document lists all the essential Visual Studio Code extensions I use for web development (MERN + JS + general productivity).

---

## UI / Themes
| Extension | Description |
|------------|--------------|
| **GitHub Dark High Contrast** | Dark theme with high contrast for better visibility |
| **Material Icon Theme** | Beautiful and consistent file & folder icons |

---

## Code Formatting & Linting
| Extension | Description |
|------------|--------------|
| **Prettier - Code Formatter** (`esbenp.prettier-vscode`) | Automatically formats JavaScript, HTML, and CSS |
| **ESLint** (`dbaeumer.vscode-eslint`) | Identifies and fixes code style issues |
| **Prettier + ESLint for CSS** (`rvest.vs-code-prettier-eslint`) | Linting + formatting for CSS files |

---

## Web Development
| Extension | Description |
|------------|--------------|
| **Live Server** (`ritwickdey.LiveServer`) | Launch local development server with live reload |
| **Auto Rename Tag** | Renames paired HTML tags automatically |
| **HTML CSS Support** | Provides CSS class and id IntelliSense in HTML |
| **Path Intellisense** | Auto-completes filenames and paths |

## Formatter Settings Summary
- **Default formatter:** Prettier  
- **Auto-save:** After delay  
- **Auto-update imports:** On file move  
- **Copilot:** Disabled globally  
- **Secondary sidebar:** Hidden by default
